# 📚 Smart Library RFID Management System

An advanced asset tracking and management system designed for modern libraries, utilizing **RFID technology** for seamless operations.

## 🚀 Key Features
* **Real-time Monitoring:** Live tracking of assets and their availability.
* **System Security:** Integrated RFID-based admin authentication.
* **Activity Logs:** Detailed history of all transactions (Borrowed/Returned) with precise timestamps.
* **Responsive Dashboard:** A modern, glassmorphism UI built with Tailwind CSS.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+).
* **Backend:** Node.js / Express (Hosted on Vercel).
* **Hardware:** RFID MFRC522, ESP32, LCD Display Screen.

## 📡 Live System Status
The dashboard connects to a live backend to provide "System Active" or "Locked" states in real-time based on admin card authentication.
